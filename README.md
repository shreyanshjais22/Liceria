**🚀 Introducing Liceria – My Full‑Stack Online Grocery Store! 🛒🥦**  

I’m thrilled to share the launch of **Liceria**, a complete e‑commerce platform that brings a modern grocery‑shopping experience to the web. From a sleek UI to real‑time order management, every piece was built from the ground up.

---

### ✨ What Liceria Offers  

- **Responsive UI** – React + TailwindCSS for a fast, mobile‑first shopping experience.  
- **Robust Cart & Checkout** – Persistent cart, real‑time stock validation, Cash‑on‑Delivery **or** Stripe payments.  
- **Order Tracking & Profile Management** – Users can view order status, manage addresses, and update their profiles.  
- **Seller Dashboard** – Add products, control inventory, and update order status instantly.  
- **Cloud‑Based Media** – Image uploads via Cloudinary (auto‑optimised) and email notifications through EmailJS.  

---

### 🛠️ Tech Stack & Architecture  

| Layer | Technology |
|-------|------------|
| **Frontend** | React, TailwindCSS, Vercel (static hosting) |
| **Backend** | Node.js, Express.js (REST API) |
| **Database** | MongoDB (scalable document store) |
| **Auth** | JWT + secure session handling |
| **Payments** | Stripe (checkout & webhook integration) |
| **Media** | Cloudinary (image CDN) |
| **Email** | EmailJS (transactional emails) |
| **Deployment** | Vercel (frontend) + Vercel Serverless Functions (API) |

---

### 🔧 Key Challenges & How I Solved Them  

| Challenge | Solution |
|-----------|----------|
| **Persistent Cart & Stock Sync** | Saved cart in localStorage + server‑side validation before checkout. |
| **Stripe Integration** | Implemented Stripe Checkout + webhook to confirm payment and auto‑create orders. |
| **Order Lifecycle Management** | Designed flexible Mongoose schemas for users, products, and order states (pending → shipped → delivered). |
| **Image Optimization** | Leveraged Cloudinary transformations for on‑the‑fly resizing & format conversion. |
| **Responsive UI** | Tailwind utility‑first approach + custom components for smooth mobile navigation. |
